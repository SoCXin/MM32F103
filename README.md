## [描述](https://github.com/SoCXin/MM32F103/wiki)

[MM32F103](https://github.com/SoCXin/MM32F103) 本产品包含 1个 12 位的 ADC、2 个比较器、2 个 16 位通用定时器、2 个 32 位通用定时器、2 个 16 位基本定时器、2 个 16 位高级定时器。

主要特点包括大量的通信接口：

* USBOTG 接口+ CAN接口
* SDIO 接口
* 7-8 个UART接口
